# 📊 Análise de Gastos com Python

Projeto simples de análise de dados utilizando Python.

## 📌 O que foi feito:
- Cálculo de média com sum() e len()
- Identificação de gastos acima de R$3000
- Cálculo de porcentagem

## 📈 Resultados:
- Média: R$ 3.144,22
- 70% dos gastos acima de R$3000

## 🚀 Tecnologias:
- Python
